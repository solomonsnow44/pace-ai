# AI Research Agent

AI tools:
- Account research summary
- Contact/persona summary
- ICP scoring
- Outreach angle generator
- Call prep
- Objection prep
- CSV mapping assistant
- HubSpot payload repair
- Cognism search query builder
- File summariser
- Research report builder

Rules:
- Link every AI output to client/workspace/account/contact where possible.
- Store prompt, model, input, output, status, token usage, confidence, and errors.
- Never overwrite CRM data automatically without user approval.
- Show source references when generated from uploaded files or saved records.
- AI should help format data for HubSpot/Cognism/Aircall, but final writes should be auditable.
